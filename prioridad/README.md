# Priority Scheduling

## Descripción

Este proyecto implementa el algoritmo:

- Priority Scheduling

El scheduler ejecuta primero el proceso con mayor prioridad.

---

# Tipo

## Puede ser:

- Apropiativo
- No Apropiativo

En esta simulación se utiliza:

- No apropiativo

---

# Funcionamiento

Cada proceso posee:

- Burst Time
- Prioridad

Menor número:

```txt
1 = mayor prioridad
```

---

# Ejemplo

```txt
P1 -> Priority 3
P2 -> Priority 1
P3 -> Priority 2
```

Orden de ejecución:

```txt
P2 → P3 → P1
```
